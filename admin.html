Bulk add players

One player per line: First,Last,Position,Nationality,Era,Age,BirthDate,Appearances
Position must be GK, DF, MF or FW. Everything after that is optional — leave blank if unsure. BirthDate as YYYY-MM-DD (e.g. 1996-04-02) — if you know it, it's better than Age since the age then updates itself forever.
Example: Jimmy,Bullard,MF,England,Premier League Era,,,

Bulk fix existing players

Same format, but this updates players already in the pool instead of adding new ones — matched by first + last name. Leave any column blank to leave that field unchanged. Use this to fix missing ages/birth dates, correct eras, or refresh appearance counts in bulk.
Example: Ahmed,Elmohamady,,,2012-2017,,,217 only changes the era and appearances, leaves everything else as-is.
